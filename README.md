# IPL Performance Analytics | SQL Project

This project focuses on analyzing Indian Premier League (IPL) batting and bowling performance data from the 2025 and 2026 seasons using SQL. The objective of the project is to transform raw cricket statistics into meaningful insights by applying various SQL concepts commonly used in Data Analytics and Business Intelligence roles.

The project uses four datasets containing batting and bowling records across two IPL seasons. Through structured querying and analysis, player performances were evaluated based on key metrics such as runs scored, batting average, strike rate, wickets taken, and overall contribution to the team.

Several business questions were addressed during the analysis. These included identifying the top run scorers and wicket takers, calculating cumulative runs across seasons, ranking players using window functions, comparing player performance against season averages, and identifying high-performing all-rounders through combined batting and bowling analysis.

The project demonstrates practical implementation of SQL concepts including SELECT statements, filtering with WHERE clauses, sorting using ORDER BY, aggregate functions such as SUM() and AVG(), GROUP BY operations, INNER JOINs, UNION ALL, subqueries, Common Table Expressions (CTEs), and window functions such as DENSE_RANK() and ROW_NUMBER().

A total of 11 analytical queries were developed to generate insights and answer performance-related questions. These queries help evaluate player consistency, seasonal rankings, and overall contribution across multiple IPL seasons.

This project was developed using MySQL Workbench and serves as a portfolio project showcasing SQL proficiency, analytical thinking, and data reporting skills. It demonstrates the ability to work with structured datasets, perform exploratory data analysis, and derive actionable insights from sports performance data.

The project is particularly relevant for aspiring Data Analysts, MIS Executives, Reporting Analysts, and Business Intelligence professionals seeking to demonstrate practical SQL skills through a real-world analytics use case.
